# ab-test-simulation

•	Simulated a two-variant landing page A/B test on 10,000 users, applying two-sample t-tests and chi-square tests at 95% confidence, confirming Variant B with a 17% higher conversion rate (p < 0.05).

•	Deployed an interactive Streamlit dashboard enabling non-technical stakeholders to explore test results, confidence intervals, and power analysis in real time.

•	Authored a business recommendation memo linking statistical output to projected revenue impact — Variant B adoption projected to yield a 12–15% uplift in monthly conversions — bridging data science findings with executive decision-making.
